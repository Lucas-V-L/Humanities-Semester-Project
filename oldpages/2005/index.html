What was flash?

Flash was revolutionary, while there were alternatives to flash, they were nowhere near what flash could do [2]. Flash allowed developers to make their websites much more interactive which greatly improved accessability to average people who did not know how to use the internet very well. While the world wide web was gaining popularity, it was still a rather new thing, and for many very unknown [source]. Flash made it much easier to use the internet as now not only was it possible to add arrows and tutorials on how to navigate your page, but once the page was loaded you no longer had to wait for new elements to load in every time you clicked something. [source] this also improved the average retention rate as not having to wait for a page to repeatedly load meant using the web was a much more pleasent experience.

Flash also saw use in the educational field, where it was used to increase students ability to retain information as it made the web easier, and more fun to use, alongside forcing user engagement to keep a student from getting distracted from the content[1]

How did it impact the web?

Flash set a new standard for the internet. it defined an age of interactive content, and if a page was not interactive, it was seen as worse than many others. Flash also set up the advancement of JavaScript, as many of the issues with Flash became more apparent, people set out for a better solution, and while JavaScript had been around for a long time already, much of the development was pushed by the popularity of flash

Flash also opened the floodgates for much more creative expression, allowing designers to escape the bounds and limits of what conventional web development allowed to create more creative, impactful, and colorful websites[2]. websites were no longer just a set of documents, but an experience, one could spend hours browsing the web, finding interesting pages made by interesting people trying to express their interesting ideas. for the first time ever, the web seemed limitless.A great example was the Levi's 150'th anniversary website. What were the drawbacks?

Flash did not come without it's drawbacks. for one, the content was non responsive [source]. this meant that content in a flash app could not easily be scaled to different screen sizes, decreasing accessebility for many users. had a large screen? get out a magnifying glass. had a small screen? have fun scrolling around to find the right button. another aspect that affected accessebility was Flash's inability to properly integrate with screenreadeader tools. [source] this hurt accessebility greatly as there were many sites built completely in flash such as the Space Jam Website. Flash also came with a plethora of security side effects, mainly because Adobe Flash player was not built in to the browser, but rather a standalone application that would run any code presented to the browser without question [source].
