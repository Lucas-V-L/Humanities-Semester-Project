Welcome to the Internet of 1995!

The internet is still new, unknown, and confusing. Not many people truly understand quite what it is. The web was originally intended as a way to exchange documents, and thats what it still does.[2] The internet is powered by a document format called HTML, standing for HyperText Markup Language, built for HTTP, the HyperText Transmision Protocol. At the moment there is very little in the way of interactive technology, which means two things. while it is less accessible to the average person, it does actually improve accessibility for disabled persons using a screenreader[6], however, screenreaders were uncommon at the time, and the real issue was getting regular people on the internet. one big leap was the advent of CSS, also known as Cascading Style Sheets. as the name implies, CSS allowed for the styling of HTML meaning web developers could get more creative with their websites, creating easy to use side navigation menus or flashy click me! buttons.[8] HTML without CSS is simply black text on a white background and nothing more, with CSS however, you get what you are viewing right now. CSS also made the web more appealing to creators who could now put creativity into their pages. It incentivised people to make more websites, meaning there was more of a reason for people to visit the web, and more of a reason for people to check out what the internet even is. This increase in visual appeal through CSS made the internet much more appealing and accessible for the average user [5].

Java Applets were also intoduced at this time.[4] Shipped with release 1.0 of Java, Java Applets were a way for website developers to embed full fledged Java applications into their websites.[9] This was the technology which the very first version of the hit game Minecraft was built atop before it transitioned to focus on desktop Java [7]. While Java applets saw moderate success at the time, they were not yet widely used as they required a special browser to run, and were more of a "proof of concept" for an interactive web than anything else[3].
